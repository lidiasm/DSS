Tipos de navegación

Navegación Implícita

Utilizando la salida JSF: JSF se encarga directamente de buscar la página que más se ajuste.

Utilizando un "Managed Bean": mediante un bean se devolverá el nombre de la página que JSF buscará igual que en el caso anterior.

Navegación Condicional: propones el id de la página que quieras abrir como parámetro con el nombre de "pageId" y en value el nº del id.
